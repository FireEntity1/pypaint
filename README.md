# PyPaint!

A basic paint program made using PyGame

Clone the repository and run the Python file!
The only required dependancy is PyGame :)

## Keybinds

- 0/1/2/3 - Switch colours
- S - Switch brush
- E - Toggle eraser
- Backspace - Clear Drawing
- Space - Save drawing as JPG
